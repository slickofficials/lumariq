# Lumariq Dispatch Service (Go + Gin + Fx)

## Install into your repo

1. From your project root, remove the old `services/dispatch-service` folder (or back it up).
2. Create a new `services/dispatch-service` folder.
3. Copy the structure and files from ChatGPT into it (keeping paths exactly the same).

4. From the project root, rebuild:

```bash
docker compose build dispatch-service --no-cache
